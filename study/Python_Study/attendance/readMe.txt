- 계정 설정
  1. data 폴더의 config.ini 파일 편집
  2. account 섹션의 id, pw 값을 설정

- PC 자동 종료 설정
  1. data 폴더의 config.ini 파일 편집
  2. options 섹션의 pcOnOff 값을 off 로 설정
     * on 으로 설정하면 PC 자동 종료 기능을 사용하지 않음

- 휴일 설정
  1. data 폴더의 config.ini 파일 편집
  2. holidayStart 값을 설정
  3. holidayEnd 값을 설정
     * 휴일이 하루일 경우 Start, End 값을 같은 날짜로 지정
     * 휴일로 지정한 날에 포함되면 자동 출퇴근 기능 동작하지 않음
